AI Bias Auditor - Important Files and Run Notes
===============================================

GitHub repository:
https://github.com/R3ap3r7/ai-bias-auditor

Do not paste or commit real API keys into source files. Put your Gemini key in a local .env file:

GEMINI_API_KEY=your-rotated-google-ai-studio-key
GEMINI_MODEL=gemini-2.5-flash

Because an API key was shared in chat, rotate it in Google AI Studio before using it again.
Gemini is used only for the final plain-English analysis report. Model selection and hyperparameter tuning run locally.

Local Run
---------
1. uv venv --python 3.12
2. source .venv/bin/activate
3. uv pip install -r requirements.txt
4. cp .env.example .env
5. Edit .env and add your Gemini key.
6. python scripts/download_demos.py
7. uvicorn app.main:app --reload
8. Open http://127.0.0.1:8000
9. For an external model audit, upload a trusted .joblib, .pkl, or .pickle file after selecting "Use uploaded model".

Validation
----------
pytest -q
python -m compileall app scripts tests

Key Files
---------
app/main.py
FastAPI app entry point. Defines the landing page, health check, demo dataset API, CSV upload API, audit API, and PDF report download API.

app/audit.py
Core bias audit engine. Handles CSV cleaning, missing values, outcome normalization, data-only pre-audit, representation balance, proxy detection, local hyperparameter tuning, tuned model comparison, uploaded-model prediction audits, Fairlearn metrics, feature importance, mitigation simulation, severity scoring, and Gemini report generation.

app/demo_data.py
Demo dataset registry and loader for COMPAS, UCI Adult, and German Credit. Requires the downloaded CSVs in data/demos and does not generate synthetic fallback data.

app/report.py
Builds downloadable PDF reports with ReportLab.

app/templates/index.html
Single-page HTML interface for upload, demo selection, column configuration, result sections, report text, and report download.

app/static/app.js
Frontend behavior for loading demos, uploading CSVs, uploading model artifacts, selecting protected/outcome columns, running separate pre-audits and post-model audits, rendering tuned model comparison, and rendering results.

scripts/download_demos.py
Downloads and preprocesses COMPAS, UCI Adult, and German Credit demo CSVs into data/demos.

tests/test_audit.py
Regression tests for outcome normalization, representation/proxy/bias detection, tuned model comparison, supported model pipelines, PDF generation, severity scoring, pre-audit API flow, uploaded model API flow, prediction validation, and upload/audit API flow.

requirements.txt
Runtime and test dependencies.

pyproject.toml
Project metadata and pytest configuration.

Dockerfile
Cloud Run compatible container definition.

.dockerignore
Files excluded from container builds.

.gitignore
Files excluded from git, including .env, .venv, caches, and generated output.

.env.example
Safe template showing which environment variables are needed for Gemini.

data/demos/compas.csv
Preloaded COMPAS demo dataset.

data/demos/adult.csv
Preloaded UCI Adult demo dataset.

data/demos/german_credit.csv
Preloaded German Credit demo dataset.

Known Notes
-----------
Gemini is optional at runtime. If GEMINI_API_KEY is missing or Gemini fails, the app shows an explicit Gemini configuration/failure message instead of pretending an AI-written report was generated.
Gemini is not used for choosing models. "Compare All Tuned Models" runs local GridSearchCV searches and ranks candidates by balanced accuracy penalized by the largest demographic parity and equalized odds gaps.
The UI uses Tailwind CDN for MVP speed. For production hardening, replace it with a compiled Tailwind build.
Uploaded pickle/joblib models can execute code when loaded. Only upload trusted local model artifacts.
